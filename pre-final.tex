\begin{frame}
  \frametitle{What is this}

  我们希望能够用 Async 和 Future 来完成内核中的并行工作，包括：
  \begin{itemize}
    \item 处理同步互斥
    \item 驱动用户线程
    \item 处理中断
    \item 实现多核 CPU 支持
  \end{itemize}
\end{frame}

\begin{frame}
  \frametitle{Why zCore}

  \begin{itemize}
    \item 目标相同：在 async rust 的基础上构建系统内核
    \item 思路相同：使用 future 的概念替代用户进程/线程
    \item 上手较易：处于初级活跃开发阶段，而且还没有 SMP 支持
  \end{itemize}
\end{frame}
